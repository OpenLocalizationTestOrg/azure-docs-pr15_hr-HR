Ovaj članak sadrži ogledne konfiguracije za konfiguriranje za Windows VMs Azure proširenja VM.

Dodatne informacije o ovim nastavcima, potražite u članku [Pregled proširenja za Azure VM.](virtual-machines-windows-extensions-features.md)

Da biste saznali više o omogućeno kućni broj, pogledajte [za izradu predložaka za nastavak.](virtual-machines-windows-extensions-authoring-templates.md)

U ovom se članku popis očekivanih konfiguracijskih vrijednosti za neke od proširenja za Windows.

## <a name="sample-template-snippet-for-vm-extensions-with-iaas-vms"></a>Isječak predložak uzorka za VM proširenja s IaaS VMs.
Isječak predložak za implementaciju proširenja izgleda kao sljedeće:

      {
      "type": "Microsoft.Compute/virtualMachines/extensions",
      "name": "MyExtension",
      "apiVersion": "2015-05-01-preview",
      "location": "[parameters('location')]",
      "dependsOn": ["[concat('Microsoft.Compute/virtualMachines/',parameters('vmName'))]"],
      "properties":
      {
      "publisher": "Publisher Namespace",
      "type": "extension Name",
      "typeHandlerVersion": "extension version",
      "autoUpgradeMinorVersion":true,
      "settings": {
      // Extension specific configuration goes in here.
      }
      }
      }

## <a name="sample-template-snippet-for-vm-extensions-with-vm-scale-sets"></a>Isječak predložak uzorka za VM proširenja s VM skaliranje skupovima.

    {

### <a name="customscript-extension-14"></a>Nastavak CustomScript 1.4.
      {
          "publisher": "Microsoft.Compute",
          "type": "CustomScriptExtension",
          "typeHandlerVersion": "1.4",
          "settings": {
              "fileUris": [
                  "http: //Yourstorageaccount.blob.core.windows.net/customscriptfiles/start.ps1"
              ],
              "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1"
          },
          "protectedSettings": {
            "storageAccountName": "yourStorageAccountName",
            "storageAccountKey": "yourStorageAccountKey"
          }
      }

#### <a name="parameter-description"></a>Opis parametra:

- fileUris: zarez odvojeni popis URL-ova datoteka koje će se preuzeti na na VM nastavka. Datoteke se preuzimaju ako ništa nije naveden. Ako datoteke Azure prostora za pohranu, u fileURLs može biti označene kao privatne i correspoding storageAccountName i storageAccountKey možete proslijediti kao privatne parametri za pristup tim datotekama.
- commandToExecute: [obavezno parametar]: to je naredba koja će se izvršavati nastavka.
- storageAccountName: [neobavezan parametar]: naziv računa za pohranu za pristup fileURLs, ako su označene kao privatne.
- storageAccountKey: [neobavezan parametar]: ključ za račun za pohranu za pristup fileURLs, ako su označene kao privatne.

### <a name="customscript-extension-17"></a>Nastavak CustomScript 1.7.

Pogledajte CustomScript verzija 1.4 za opis parametra. Verzija 1.7 predstavlja podrška za slanje skripte parameters(commandToExecute) kao protectedSettings, u tom se slučaju oni će biti šifrirana prije slanja. parametar 'commandToExecute' se može odrediti postavke ili protectedSettings ali ne i u oba.

        {
            "publisher": "Microsoft.Compute",
            "type": "CustomScriptExtension",
            "typeHandlerVersion": "1.7",
            "settings": {
                "fileUris": [
                    "http: //Yourstorageaccount.blob.core.windows.net/customscriptfiles/start.ps1"
                ],
                "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1"
            },
            "protectedSettings": {
              "commandToExecute": "powershell.exe -ExecutionPolicy Unrestricted -start.ps1",
              "storageAccountName": "yourStorageAccountName",
              "storageAccountKey": "yourStorageAccountKey"
            }
        }
