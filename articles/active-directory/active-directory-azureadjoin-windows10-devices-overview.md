# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>Windows 10 za tvrtke: načini korištenja uređaji za posao

Windows 10 omogućuje vam odražava Azure Active Directory (Azure AD). Uređaje Windows 10 možete povezati Azure AD da bi korisnici mogli prijaviti u sustav Windows pomoću računa za Azure AD ili dodavanjem identifikacijskim Azure da biste pristupili poslovnih aplikacija i resursa.
