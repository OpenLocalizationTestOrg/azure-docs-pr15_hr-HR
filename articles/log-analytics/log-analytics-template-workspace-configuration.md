# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Upravljanje zapisnika analize pomoću predložaka Voditelj resursa za Azure

Možete koristiti [Voditelj resursa Azure predloške] (.. / azure-resource-manager/resource-group-authoring-templates.md) za stvaranje i konfiguriranje radnih prostora za zapisnik analize. Primjeri zadatke koje možete izvršiti s predlošcima obuhvaćaju sljedeće:

+ Stvaranje radnog prostora
+ Dodavanje rješenja
+ Stvaranje spremljenih pretraživanja
+ Stvaranje grupe za računala
+ Omogućivanje zbirke IIS zapisnika s računala s instaliran agent za Windows
+ Prikupljanje mjerača performansi s računala Linux i Windows
+ Prikupite događaje iz syslog na računalima Linux 
+ Prikupite događaje iz zapisnika događaja sustava Windows
+ Prikupljanje zapisnika prilagođene događaja
+ Dodavanje analize agent zapisnika Azure virtualnog računala
+ Konfiguriranje zapisnika analize indeksirati podatke koji se prikupljaju korištenjem dijagnostike Azure


Ovaj članak sadrži predložak primjere koji pokazuju neke konfiguracije koje možete izvršiti iz predložaka.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Stvaranje i konfiguriranje zapisnika analize radnog prostora

Sljedeća Ogledna predložak ilustrira način:

1.  Stvaranje radnog prostora
2.  Dodajte rješenja u radni prostor
3.  Stvaranje spremljenih pretraživanja
4.  Stvaranje grupe za računala
5.  Omogućivanje zbirke IIS zapisnika s računala s instaliran agent za Windows
6.  Prikupljanje mjerača mjerača performansi logičkog diska s računala Linux (% Inodes koristi Besplatni megabajta; % Koristi prostor; Na disku prijenosi/sec; Na disku čitanja/sec; Zapisivanje na disku/sec)
7.  Prikupite syslog događaje iz Linux računala
8.  Prikupljanje pogrešaka i upozorenja događaje iz zapisnik događaja aplikacije s računala sa sustavom Windows
9.  Prikupljanje memorije dostupne Mbytes performanse brojač s računala sa sustavom Windows
10. Prikupljanje prilagođene zapisnika 
11. Prikupljanje zapisnika IIS i zapisnike događaja sustava Windows Azure Dijagnostika napisao s računom za pohranu

### <a name="deploying-the-sample-template"></a>Implementacija oglednog predloška

Uvođenje predloška uzorak:

1. U datoteci, primjerice spremili priložene uzorka`azuredeploy.json` 
2. Uređivanje predloška imati konfiguraciju želite
3. Uvođenje predloška pomoću programa PowerShell ili naredbenog retka

#### <a name="powershell"></a>PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Naredbeni redak

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
