## <a name="how-to-create-the-nsg-for-the-front-end-subnet"></a>Kako stvoriti NSG za podmreže sučelje
Da biste stvorili programa NSG pod nazivom imenovani **NSG sučelju** ovisno o scenariju iznad, slijedite ove korake:

1. Ako još niste koristili Azure PowerShell, pročitajte članak [Instaliranje i konfiguriranje Azure PowerShell](../powershell-install-configure.md) i slijedite upute do kraja do kraja se prijaviti u Azure i odaberite svoju pretplatu.

3. Stvaranje sigurnosne grupe mreže pod nazivom **NSG sučelju**.

        New-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" -Location uswest `
            -Label "Front end subnet NSG"

    Očekivani izlaz:

        Name         Location   Label               
        ----         --------   -----               
        NSG-FrontEnd West US    Front end subnet NSG


4. Stvorite pravilo sigurnost dopuštanja pristupa s Interneta priključak 3389.

        Get-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" `
        | Set-AzureNetworkSecurityRule -Name rdp-rule `
            -Action Allow -Protocol TCP -Type Inbound -Priority 100 `
            -SourceAddressPrefix Internet  -SourcePortRange '*' `
            -DestinationAddressPrefix '*' -DestinationPortRange '3389'

    Očekivani izlaz:

        Name     : NSG-FrontEnd
        Location : Central US
        Label    : Front end subnet NSG
        Rules    :

                      Type: Outbound

                   Name                 Priority  Action   Source Address  Source Port   Destination      Destination    Protocol
                                                           Prefix          Range         Address Prefix   Port Range

                   ALLOW VNET OUTBOUND  65000     Allow    VIRTUAL_NETWORK *             VIRTUAL_NETWORK  *              *       
                   ALLOW INTERNET       65001     Allow    *               *             INTERNET         *              *       
                   OUTBOUND                                                                                                      
                   DENY ALL OUTBOUND    65500     Deny     *               *             *                *              *

4. Stvorite pravilo sigurnost dopuštanja pristupa s Interneta priključak 80.

        Get-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" `
        | Set-AzureNetworkSecurityRule -Name web-rule `

## <a name="how-to-create-the-nsg-for-the-back-end-subnet"></a>Kako stvoriti NSG za podmreži pozadinskih
3. Stvaranje sigurnosne grupe mreže pod nazivom **NSG pozadinskog**.

        New-AzureNetworkSecurityGroup -Name "NSG-BackEnd" -Location uswest `
            -Label "Back end subnet NSG"

    Očekivani izlaz:

        Name        Location   Label              
        ----        --------   -----              
        NSG-BackEnd West US    Back end subnet NSG


4. Stvorite pravilo sigurnost dopuštanja pristupa iz podmreže sučelje priključak 1433 (zadani je priključak koristi SQL Server).

        Get-AzureNetworkSecurityGroup -Name "NSG-FrontEnd" `
        | Set-AzureNetworkSecurityRule -Name rdp-rule `
            -Action Allow -Protocol TCP -Type Inbound -Priority 100 `
            -SourceAddressPrefix 192.168.1.0/24  -SourcePortRange '*' `
            -DestinationAddressPrefix '*' -DestinationPortRange '1433'

    Očekivani izlaz:

        Name     : NSG-BackEnd
        Location : Central US
        Label    : Back end subnet NSG
        Rules    :

                   ALLOW VNET INBOUND   65000     Allow    VIRTUAL_NETWORK *             VIRTUAL_NETWORK  *              *

4. Stvorite pravilo sigurnost blokiranje pristupa podmreži s Internetom.

        | Set-AzureNetworkSecurityRule -Name block-internet `

            -SourceAddressPrefix '*'  -SourcePortRange '*' `
